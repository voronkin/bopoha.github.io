Кто мы?

Группа компаний Инфраторг, официальный дилер российского производителя инфракрасных обогревателей нового поколения «ПИОН», предлагает решения для экономного электрического обогрева любых помещений. Обогрев квартиры, дома, дачи, веранды, гаража. Для малого и среднего бизнеса обогрев любых инфраструктурных объектов — кафе, ресторанов, бытовок, автосервисов, складов и ангаров. Для животноводов — обогрев курятников, свинарников, коровников, кроликов. Позвоните нам, чтобы узнать о специальных предложениях!
